# VibePM

Start a structured project definition session.

## Instructions

1. Read the PM Skill definition file at `VibePM/SKILL.md`
2. Read all reference files in `VibePM/references/` for detailed frameworks
3. Follow the Core Dialogue Loop defined in SKILL.md
4. Maintain the project profile throughout the conversation
5. When ready, generate output using templates from `VibePM/references/output-templates.md`

## Quick Start

Tell me about your project idea. It can be as vague as "I want to build an AI product" or as specific as a full project brief that needs review.

I will guide you through a structured conversation to:
- Clarify your project goals and target users
- Define the problem you're solving
- Converge on a clear scope and MVP
- Identify risks and constraints
- Generate a professional project definition document

## Usage

This workflow supports three modes:
- **From scratch**: You have a vague idea
- **Refine existing**: You have a plan but it needs clarity
- **Unblock**: Your project is stuck and needs re-alignment
