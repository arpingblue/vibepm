Start a project definition session using the PM Skill.

Read the skill definition at VibePM/SKILL.md and all reference files in VibePM/references/ to understand the complete framework.

Then follow the Core Dialogue Loop:
1. Summarize current understanding
2. Identify information gaps
3. Ask 2-4 high-value questions
4. Explain why you're asking
5. Decide whether to continue or generate output

Begin by asking the user about their project idea. Support three modes:
- Mode 1: Build from scratch (vague idea)
- Mode 2: Refine existing (messy plan)
- Mode 3: Unblock (stuck project)

When ready, use templates from VibePM/references/output-templates.md to generate deliverables.
